# Nullbank

NullBank é uma maquininha de cartão de crédito pra você que ta sem dinheiro irmão

## Instalando

Digite: 
```
git clone https://github.com/mateuschaves/NullBank.git
```
## Pré-requisitos

[Arduino IDE](https://www.arduino.cc/en/main/software)
